# Vocab Quiz — Green

단일 HTML/CSS/JS로 동작하는 가벼운 영단어 퀴즈 웹앱입니다. GitHub Pages에 그대로 올려서 사용할 수 있습니다.

## 기능
- 테마 선택, 모드 선택(영→한 / 한→영)
- 테마 내 단어 무작위 출제, 정답 제출/표시, 다음 문제 이동(Enter 키 지원)
- 진행도 표시 (현재/전체)
- 완료 후 틀린 문제만 다시 풀기 또는 처음으로 이동

## 빠른 시작
1. 이 저장소를 클론하거나 ZIP을 다운로드해 그대로 열어보세요.
2. 로컬에서 테스트하려면 `index.html`을 브라우저로 열면 됩니다(Chrome 권장).  
   ※ 일부 브라우저는 `file://` 경로에서 `fetch`를 제한합니다. 그럴 경우 아래처럼 간단 서버를 띄우세요.
   ```bash
   # Python 3
   python -m http.server 8080
   # 또는
   npx http-server . -p 8080
   ```
   이후 `http://localhost:8080` 접속.
3. 배포는 GitHub Pages를 사용하세요.
   - 새 리포지토리 생성 → 이 프로젝트 파일 업로드
   - Settings → Pages → Deploy from a branch → `main` / `/ (root)` 선택 → 저장
   - 잠시 후 페이지가 활성화됩니다.

## 단어장 편집(가장 중요)
`data/vocab.json` 파일을 열어 내용을 수정하세요. 포맷:
```json
{
  "themes": {
    "Animals": [
      { "en": "cat", "ko": "고양이" },
      { "en": "tiger", "ko": ["호랑이", "범"] }
    ],
    "Daily": [
      { "en": "desk", "ko": "책상" }
    ]
  }
}
```
- `en` / `ko`에 문자열 또는 문자열 배열을 넣을 수 있습니다(배열이면 모든 항목이 정답으로 인정).
- **모드**는 앱에서 선택합니다(영→한, 한→영 모두 지원).

## 키보드/접근성
- Enter: 정답 제출 또는 다음 문제 이동(상태에 따라 자동 전환)
- 스크린리더용 라벨/라이브영역 추가

## 커스텀 스타일
- `styles.css`에서 색상(초록 계열)과 간격을 조정할 수 있습니다.
- 폰트는 Pretendard(구글폰트) 로드, 시스템 폰트로 자동 폴백됩니다.
