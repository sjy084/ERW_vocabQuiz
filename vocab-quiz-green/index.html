단어장 파일(vocab.json) 교체/수정 방법

프로젝트 루트의 data/vocab.json 파일을 수정하세요. 포맷은 아래 예시를 참고하세요.

{
  "themes": {
    "Animals": [
      { "en": "cat", "ko": "고양이" },
      { "en": "dog", "ko": "개" },
      { "en": "tiger", "ko": ["호랑이", "범"] }
    ],
    "Daily": [
      { "en": "desk", "ko": "책상" }
    ]
  }
}

여러 정답을 허용하려면 문자열 대신 배열을 사용하면 됩니다(위의 tiger 참조).
